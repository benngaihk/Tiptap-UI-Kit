## [0.1.0] - 2024-12-14

### Added
- Initial release
- Core editor component (TiptapProEditor)
- 45+ Vue components
- 5 theme presets (Default, Word, Notion, GitHub, Typora)
- Light/Dark mode support
- Word document mode with A4 layout and pagination
- AI features:
  - Continue writing
  - Polish text
  - Translation (14+ languages)
  - Summarize
  - Custom AI prompts
- Real-time collaboration with Yjs
- i18n support (Chinese Simplified/Traditional, English)
- Rich text editing features:
  - Text formatting (bold, italic, underline, etc.)
  - Headings (H1-H6)
  - Lists (ordered, unordered, task)
  - Tables with full toolbar
  - Code blocks with syntax highlighting
  - Images with resize
  - Links
  - Colors and highlights
  - Font family and size
  - Text alignment
  - Subscript/Superscript
  - Format painter
- Advanced tools:
  - Drag handle menu
  - Floating menu
  - Link bubble menu
  - Table toolbar
  - Image toolbar
  - Zoom control
  - Preview mode
  - Device switcher
- Modular architecture with feature flags
- Preset configurations (minimal, basic, advanced, full)
- TypeScript support
- Comprehensive styling system with CSS variables

### Fixed
- Toolbar positioning in fixed mode
- Drag handle menu styling
- Various UI/UX improvements

## [0.0.1] - 2024-11-01

### Added
- Project initialization

## Release Types

- **Major** (x.0.0): Breaking changes
- **Minor** (0.x.0): New features, backwards compatible
- **Patch** (0.0.x): Bug fixes, backwards compatible

## How to Update

### For npm users:
```bash
npm update tiptap-ui-kit
```

### For pnpm users:
```bash
pnpm update tiptap-ui-kit
```

### For yarn users:
```bash
yarn upgrade tiptap-ui-kit
```
